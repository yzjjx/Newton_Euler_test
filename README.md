# 关于牛顿欧拉法的尝试
## 文件描述
trans_matrix.m：用于生成符号化的T矩阵，主要公式依据《机器人学导论》P53
trans_matrix.m：用于生成速度与角速度的递推公式，递推公式按照《机器人学导论》P102、P103，公式(5.45)、公式(5.47)
newton_euler_external.m：牛顿欧拉外推，主要公式依据《机器人学导论》P124，公式(6.45)到(6.50)
newton_euler_internal.m：牛顿欧拉外推+内推，主要公式依照《机器人学导论》P124，验证依照P125-P127
test1.m：对一些matlab符号计算功能的验证
